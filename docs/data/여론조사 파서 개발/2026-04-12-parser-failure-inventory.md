# 2026-04-12 여론조사 파서 문제 인벤토리

## 목적

`https://dev.lawdigest.kr/election?tab=poll`에 실제로 노출된 오염 데이터를 기준으로, 문제가 있는 parser를 식별하고 `등록번호 -> parser -> 원본 PDF -> fixture -> 수정 전략`까지 바로 연결한다.

이번 문서는 전체 ingest 재실행 대신, parser별 짧은 피드백 루프를 돌리기 위한 작업 기준 문서다.

## 실제 확인된 데이터 오염

### 1. `_WinjiKoreaParser`

- 대상: `14571`
- 실제 저장 예
  - `기타 지지하는`
  - `더불어 조국 잘`
  - `국민의힘 진보당 개혁신당 다른 정당이`
  - `선택지6`, `선택지7`, `선택지8`
- 해석
  - 헤더 행에서 선택지를 제대로 분리하지 못하고, 비율 개수에 맞춰 placeholder를 뒤에 채우는 현재 로직이 그대로 드러난다.
- 수정 방향
  - 서울 PDF를 fixture로 추가
  - 포맷 A/B 선택지 추출 시 `header_row` 후보를 더 엄격히 잡고, 병합된 셀을 분리하지 못하면 질문 폐기
  - `선택지N` fallback은 유지하더라도 ingest 전 quality gate에서 반드시 차단

### 2. `_MediaTomatoParser`

- 대상: `14671`, `14730`
- 실제 저장 예
  - `민주당국민의힘`
  - `혁신당개혁신당진보당`
  - `박주민조국김민석정원오서영교강훈식전현희박홍근그 외`
  - `없음잘 모름`
- 해석
  - 텍스트 기반 컬럼 분할이 제대로 안 되어, 열 경계가 무너진 상태로 선택지가 붙는다.
- 수정 방향
  - 서울 뉴스토마토 PDF 2건을 fixture로 추가
  - 텍스트 기반 column slicing 규칙을 pollster 전용으로 재조정
  - 정당/후보 질문에서 다중 고유명사 연쇄가 나오면 질문 자체를 폐기

### 3. `_EmbrainPublicParser`

- 대상: `14964`
- 실제 저장 예
  - 질문 1 제목 공란
  - 정당지지도 옵션 `선택지1~8`
- 해석
  - 제목 행/선택지 행 인식이 어긋나 있어, 질문 제목과 헤더를 제대로 못 읽고 있다.
- 수정 방향
  - 뉴스1 서울 PDF를 fixture로 추가
  - `HEADER_ROW_INDEX=1` 전제가 깨지는 케이스를 재현하고, 제목/헤더 row 탐색 로직을 동적으로 변경
  - 헤더를 못 찾으면 질문 폐기

### 4. `_KStatResearchParser`

- 대상: `15366`
- 실제 저장 예
  - `Q1~Q11`
  - `정원오 현 성동구청`
  - `박주민 국회의`
  - `오세훈 현 장 서울시장`
- 해석
  - 제목 fallback이 너무 약해서 실제 질문 제목 복원에 실패하고, 일부 라벨은 줄바꿈 경계가 어색하게 합쳐진다.
- 수정 방향
  - 기존 경기 fixture를 유지한 채 서울 KBS PDF fixture 추가
  - KBS 포맷에서 제목 문구를 `Qn` 대신 페이지 텍스트에서 추출
  - 후보/정당 라벨의 직함 토큰 결합 규칙 보강
  - 제목 복원 실패 시 질문 폐기

### 5. `_FlowerResearchParser`

- 대상: `15891`
- 실제 저장 예
  - `먚鰃믅믅鱮 (1)`
  - `꾍랹 긁묁鰃`
- 해석
  - GID 디코딩 경로가 특정 파일에서 실패해 문자 깨짐이 발생한다.
- 수정 방향
  - 서울 PDF를 fixture로 추가
  - GID decode 전/후 텍스트 샘플을 비교하는 재현 테스트 추가
  - 복원이 안 되면 해당 질문은 ingest에서 폐기

## 작업 원칙

1. 복원 가능한 parser는 parser 자체를 수정한다.
2. 복원 불가능한 질문은 ingest 전에 폐기한다.
3. `선택지N`, 병합 문자열, mojibake, blank/Qn 제목은 화면까지 보내지 않는다.
4. 전체 지방선거 ingest 대신, parser별 PDF/fixture 재현 테스트를 우선한다.

## 즉시 실행 순서

1. `_WinjiKoreaParser` 서울 fixture 추가 및 선택지 추출 보강
2. `_MediaTomatoParser` 서울 fixture 2건 추가 및 텍스트 분리 규칙 보강
3. `_EmbrainPublicParser` 서울 fixture 추가 및 제목/헤더 row 탐색 보강
4. `_KStatResearchParser` 제목 복원 및 라벨 결합 보강
5. `_FlowerResearchParser` mojibake 재현 테스트 추가 후 복원 불가 질문 폐기
