## 7. 지역 확장 시 주의사항

파서는 기존 서울·경기 데이터로 개발되었으므로, 다른 지역 PDF에서 **동일 기관이라도 포맷이 다를 수 있음**.

### 검증 우선 지역

| 지역 | 이유 |
|------|------|
| 경상북도 (144건, 38.2%) | 미개발 기관 건수 최다 (89건) |
| 광주광역시 (62건, 24.2%) | 커버율 최저 |
| 전라남도 (196건, 41.8%) | 전체 건수 최다이나 커버율 낮음 |

### 기존 파서 검증 방법

```bash
cd services/data
# 특정 지역 PDF로 파서 동작 확인
python scripts/polls/probe_parsers.py --target <slug>

# 스크리닝 도구로 PDF 포맷 확인
python scripts/dev/diagnose_pdf.py --pdf "파일명.pdf" --pollster "기관명" --human --dump-text
```
